## 一、登录与权限

### 1.1 登录

打开系统会自动跳转到登录页。输入用户名和密码即可登录。

| 角色 | 默认账号 | 权限说明 |
|------|----------|----------|
| 管理员 | admin / admin123 | 全部功能 + 审批权限 |
| 普通用户 | 需管理员创建 | 默认无权限，需申请 |

### 1.2 普通用户申请权限

1. 用普通用户账号登录后，页面会显示灰色遮罩和🔒图标
2. 点击「**申请权限**」按钮
3. 等待管理员审批（状态显示"审批中"）
4. 审批通过后，刷新页面即可正常使用

### 1.3 管理员审批权限

1. 有用户申请时，顶栏「⚙️ 权限」按钮会显示红色角标
2. 点击进入权限管理页面
3. 看到申请列表，点击「**同意**」或「**拒绝**」

### 1.4 退出登录

点击顶栏右侧的退出图标即可。

## 二、仪表盘使用

登录后默认进入仪表盘，显示：

| 区域 | 内容 |
|------|------|
| KPI 卡片 | 物料总数 / 待处理采购 / 进行中工单 / 低库存预警 / MRP例外项 |
| 库存概览 | 横向条形图，按模块平等统计库存 |
| 工单状态 | 待下达/已下达/进行中/已完成/已关闭 |
| 低库存预警 | 低于安全库存的物料列表 |

## 三、物料管理

点击左侧菜单「**物料管理**」查看和搜索所有物料。

物料信息包含：编码、名称、规格、品牌、单价、提交人、采购链接。

---

## 四、BOM管理

点击左侧菜单「**BOM 管理**」查看产品结构。

系统已预导入「三工位测试台（15台套）」的全部 BOM 数据，包含外购件、外加工件、电气、视觉、量具工具共 6 个模块。

---

## 五、MPS主计划

点击左侧菜单「**MPS 主计划**」录入成品需求：
1. 点击「添加计划」
2. 选择成品物料（三工位测试台）
3. 填写计划日期和数量
4. 保存

---

## 六、MRP运算

1. 点击左侧菜单「**MRP 运算**」
2. 点击「一键 MRP 运算」
3. 系统自动计算所有物料的需求
4. 查看结果后可点击「一键转为采购申请」生成采购单

### 结果解读

| 列名 | 说明 |
|------|------|
| 物料编码 | 需要采购或生产的物料 |
| 建议下达日 | 应该什么时候下单 |
| 需求日期 | 物料需要在什么日期到货 |
| 数量 | 建议的采购或生产数量 |

## 七、采购管理

### 查看采购单

点击左侧菜单「**采购管理**」，查看所有采购订单。

采购单自动包含：品牌、单价、提交人、采购链接（从 BOM 数据同步）。

### 状态流转

```
申请 → 已审批 → 已下单 → 部分收货 → 已完成
```

### 需求日

采购管理页面新增「需求日」列，显示 MRP 计算的物料需求日期。

## 九、库存管理

点击左侧菜单「**库存管理**」查看所有物料的库存情况。

- 现有库存：当前可用数量
- 安全库存：低于此数会触发警告
- 入库：增加库存
- 出库：减少库存

## 十、定时MRP与邮件通知

### 10.1 设置定时MRP

1. 点击左下角「⚙️ **系统工具**」
2. 打开「启用」开关
3. 设置执行时间（如每天 06:00）
4. 系统将每天自动运行 MRP 并生成采购单

### 10.2 配置邮件通知

1. 在系统工具中展开「邮件通知」
2. 填写 SMTP 信息（以 QQ 邮箱为例）：

| 字段 | QQ邮箱填什么 |
|------|-------------|
| 收件 | `your@qq.com` |
| 服务器 | `smtp.qq.com` |
| 端口 | `587` |
| 用户名 | `your@qq.com` |
| 密码 | QQ邮箱授权码（16位） |

3. 点击「保存配置」
4. 点击「测试邮件」验证

> 📌 授权码获取：QQ邮箱 → 设置 → 账户 → POP3/SMTP服务 → 开启 → 短信验证 → 复制16位授权码

## 十一、常见问题

### Q1：登录提示"用户名或密码错误"

确认用户名和密码正确。默认管理员账号 admin / admin123。

### Q2：普通用户登录后看不到数据

普通用户默认无权限，需点击遮罩上的「申请权限」按钮，等待管理员审批。

### Q3：MRP运算没有结果

需要先在「MPS 主计划」录入成品需求计划，MRP 才能计算。

### Q4：授权码是什么？

授权码是邮箱服务商为第三方应用颁发的专用密码，不是邮箱登录密码。QQ邮箱在设置 → 账户中生成。

### Q5：定时MRP怎么不执行？

检查系统工具的「启用」开关是否打开，时间设置是否正确。

### Q6：怎么备份数据？

点击系统工具 → 导出，下载 JSON 备份文件。
