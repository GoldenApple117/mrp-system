## 一、如何录入物料信息

### 第一步：进入物料管理页面

点击左侧菜单「**物料管理**」。

### 第二步：新增物料

1. 点击左上角「**新增物料**」按钮
2. 在弹出的窗口中填写：
   - **物料编码**（必填）— 如 FG-001、RM-001，建议用前缀区分类型
   - **物料名称**（必填）— 如 智能温控器
   - **规格型号** — 如 220V/50Hz
   - **单位** — 如 台、个、块、套
   - **物料类型** — 选择成品/半成品/零件/原材料
   - **提前期(天)** — 采购或生产需要多少天
   - **安全库存** — 低于这个数量时需要补货
   - **批量规则** — LFL按需定货 / FOQ固定批量 / EOQ经济批量 / MULT倍数批量
   - **批量数量** — 批量规则对应的数值
   - **是否外购** — 是=采购件 / 否=自制件
3. 点击「**保存**」

### 第三步：搜索和筛选

- 在搜索框输入物料编码或名称，按回车搜索
- 点击物料类型下拉框，筛选指定类型的物料

### 批量操作

勾选多个物料 → 点击「**批量停用**」→ 确认即可一次性停用多个物料。

## 二、如何建立BOM（产品结构）

### 方式一：从Excel导入（推荐）

这是最快的方式，特别适合首次大批量录入。

1. 点击左侧菜单「**BOM管理**」
2. 点击「**下载模板**」按钮，系统会生成一个Excel模板
3. 按模板格式填写（模板包含示例数据供参考）
4. 点击「**导入Excel BOM**」，选择填好的Excel文件
5. 系统会自动：
   - 读取「物料主数据」工作表 → 自动创建不存在的物料
   - 读取「BOM」工作表 → 自动建立层级父子关系
6. 导入完成后会提示：`成功导入 X 个BOM，新建物料 Y 个`

### 方式二：手动创建

1. 点击「**新建BOM**」按钮
2. 填写BOM编号，选择成品物料
3. 在下方的表格中添加BOM行（父物料→子物料→用量）
4. 点击「**保存**」

### 查看BOM树形结构

在BOM列表中点击任意BOM的「**查看**」，会弹出树形图显示完整的产品结构。

### 用途反查

想知道某个物料被哪些成品用到？在BOM列表中可以查看。

### 版本管理

- **创建新版本**：基于现有BOM创建新版本，行数据会自动复制
- **激活BOM**：将某版本设为生效，同一成品的其他生效版本自动失效
- 每个成品同一时刻只有一个生效的BOM版本

### ECN工程变更

1. 提交变更申请 → 填写变更原因和内容
2. 审批 → 审批通过后BOM自动生效

## 三、如何录入MPS生产计划

### 单个添加

1. 点击左侧菜单「**MPS 主计划**」
2. 点击「**添加计划**」按钮
3. 选择成品物料、计划日期、数量
4. 点击「**保存**」

### 批量添加（推荐）

1. 点击「**批量添加**」按钮
2. 选择成品物料
3. 设置开始日期和结束日期
4. 填写每周计划数量
5. 点击「**生成**」— 系统会自动按每周一条生成该时段的所有计划
6. 例如：开始日期7月1日，结束日期9月30日，每周100台
   - 会生成 7/1、7/8、7/15……共13条计划

### 筛选和删除

- 用日期范围筛选查看指定时段内的计划
- 勾选多条计划 → 点击「**批量删除**」

> **提示**：MPS只需录入成品计划，MRP引擎会自动展开计算到所有原材料。

## 四、如何执行MRP运算

MRP运算会读取MPS + BOM + 库存数据，自动生成所有物料的采购/生产建议。

### 操作步骤

1. 点击左侧菜单「**MRP 运算**」
2. 设置参数（通常保持默认即可）：
   - **展望期(天)** — 计划未来多少天的需求，默认90天
   - **时界(天)** — 多少天内的计划不可调整，默认7天
3. 点击「**一键 MRP 运算**」
4. 等待约1秒，系统会显示：
   - **统计摘要**：总订单数、采购建议数、生产建议数
   - **例外信息**：红色=缺料错误，黄色=安全库存告警
   - **计划订单**：详细的采购/生产建议列表

### 结果解读

| 列名 | 说明 |
|------|------|
| 物料编码 | 需要采购或生产的物料 |
| 类型 | 采购（外购件）或 生产（自制件） |
| 建议下达日 | 应该什么时候下单/下产线 |
| 需求日期 | 物料需要在什么日期到货/完工 |
| 数量 | 建议的采购或生产数量 |

## 五、如何将MRP建议转为订单

MRP运算完成后，结果页面会显示「**一键转为采购申请/工单**」按钮：

1. 确认计划订单列表中的数据无误
2. 点击该按钮
3. 系统将自动：
   - 采购建议 → 创建采购申请（状态=申请）
   - 生产建议 → 创建生产工单（状态=待下达）
4. 成功后提示转换数量

之后可分别到「采购管理」和「生产管理」页面查看和跟踪。

## 六、如何管理采购订单

### 查看采购单

点击左侧菜单「**采购管理**」，默认显示采购订单列表。

### 状态流转

采购单状态沿以下路径流转：
```
申请 → 已审批 → 已下单 → 部分收货 → 已完成
```

每个状态都可以手动更新：
1. 点击某条采购单的「更新状态」
2. 选择新状态
3. 如果是收货操作，填写收货数量

### 批量操作

勾选多个采购单 → 点击：
- 「**批量下单**」— 一批采购单同时标记为已下单
- 「**批量完成**」— 一批采购单同时标记为已完成
- 「**批量删除**」— 删除处于"申请"状态的采购单

### 供应商管理

点击「**供应商管理**」标签页，可以新增和查看供应商信息。

## 七、如何管理生产工单

### 查看工单

点击左侧菜单「**生产管理**」。

### 状态流转

```
待下达 → 已下达 → 进行中 → 已完成
```

双击工单行的「更新状态」按钮进行操作。

### 批量操作

勾选多个工单 → 点击：
- 「**批量下达**」— 下发到车间
- 「**批量开工**」— 标记为生产中
- 「**批量完工**」— 标记为已完成
- 「**批量删除**」— 删除处于"待下达"状态的工单

### 工作中心和工艺路线

可以在生产管理页面查看和创建工作中心（如装配线1号、加工中心等）。

## 八、如何管理库存出入库

### 查看库存

点击左侧菜单「**库存管理**」，页面展示：
- 物料编码/名称/库存数量/安全库存/是否偏低
- 库存总览：按物料维度汇总

### 入库操作

1. 点击某物料的「**入库**」
2. 填写入库数量和来源（如采购入库、生产入库）
3. 确认 → 库存增加

### 出库操作

1. 点击某物料的「**出库**」
2. 填写出库数量和用途（如生产领料、销售出库）
3. 确认 → 库存减少

### 查看流水

点击「出入库流水」标签页，查看所有库存变动记录。

## 九、如何做到货检验

采购的物料到货后，需要先检验再入库。

### 操作步骤

1. 点击左侧菜单「**检验盘点**」
2. 在「到货检验」标签页，点击「**创建检验**」
3. 选择对应的采购单，填写检验数量
4. 创建成功后，在列表中点击「**录入结果**」
5. 填写合格数量和不合格数量
6. 提交 → 合格部分**自动入库**，库存自动增加

### 流程说明

```
创建检验单 → 检验员实际检验 → 录入合格/不合格数量
→ 合格→自动入库（更新库存+采购单已收数量）
→ 不合格→记录不合格数量（可退货）
```

## 十、如何做库存盘点

### 操作步骤

1. 点击左侧菜单「**检验盘点**」
2. 切换到「盘点管理」标签页
3. 点击「**创建盘点**」
4. 选择物料和仓库 → 系统自动记录当前系统库存
5. 创建成功后，在列表中点击「**录入结果**」
6. 填写实际盘点的数量 → 提交 → 自动计算差异
7. 确认差异无误 → 点击「**确认调整**」→ 库存自动修正

### 举例

- 系统库存：200件
- 实盘数量：195件
- 差异：-5件
- 确认调整后：库存变更为195件

## 十一、如何查看CRP产能负荷

CRP用来检查工作中心的产能是否够用。

1. 点击左侧菜单「**CRP产能计划**」
2. 设置展望期（周数）
3. 点击「**计算产能负荷**」
4. 系统显示：
   - **综合利用率** — 整体负荷比例
   - **瓶颈工作中心** — 利用率超过100%的工作中心
   - **产能负荷明细** — 每个工作中心每周的负荷情况

> 利用率 > 100% 标记为超载（红色）
> 利用率 > 130% 标记为严重瓶颈

## 十二、如何批量操作

系统中多个页面支持批量操作，操作方法统一：

### 使用步骤

1. **勾选**表格左侧的复选框，选中要操作的条目
2. 页面顶部出现**红色批量操作栏**，显示已选数量
3. 点击需要的批量按钮（如批量删除、批量完成等）
4. 确认弹窗 → 系统逐个执行 → 完成后提示结果

### 批量操作一览

| 页面 | 可使用操作 |
|------|-----------|

| 生产管理 | 批量下达、批量开工、批量完工、批量删除 |

## 十三、常见问题

### Q1：MRP运算提示“无MPS数据”怎么办？

需要先在「MPS 主计划」页面录入成品计划。MRP的输入是MPS，没有MPS就无法计算。

### Q2：导入Excel后提示“0个BOM”是怎么回事？

检查Excel文件是否包含两个工作表：
- **BOM**（必填）— 存放父子物料关系
- **物料主数据**（推荐）— 存放物料信息

如果没有物料主数据表，需要先在系统中手动创建物料再导入。

### Q3：采购单创建后怎么无法删除？

只有处于「申请」状态的采购单可以删除。如果已经流转到「已审批」或「已下单」，需要先作废或完成。

### Q4：工单创建后怎么修改？

工单创建后不能直接修改。如果需要调整，可以：
- 删除原工单（仅「待下达」状态可删）
- 重新创建新的工单

### Q5：BOM激活后旧的版本会怎样？

激活新版本时，同一成品的其他生效版本会自动变为「失效」状态，并记录失效日期。历史版本不会被删除，可以随时查看。

### Q6：为什么订单准时率（OTD）图表中订单都显示为灰色？

灰色代表「未完成」——这些订单的需求日期还没到，属于正常状态。OTD 图表按三色分类：
- **绿色**：已完成（按时交付）
- **红色**：逾期（过了需求日期但未完成）
- **灰色**：未完成（未到需求日期，未到期）

### Q7：部署到 Railway 后我的数据安全吗？

数据部署在 Railway 托管的 MySQL 数据库中，完全独立于应用容器，**不会因重新部署而丢失**。数据库有自动备份。
