# MRP II 系统测试指南

> 让任何 AI 助手都能快速启动并测试此系统（最新版 v1.2.0）

---

## 一、一键启动

```bash
# 后端
cd mrp-system/backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

# 前端
cd mrp-system/frontend
npm install && npm run dev
```

**访问地址：** 前端 `http://127.0.0.1:5173` | API文档 `http://127.0.0.1:8000/docs`

## 二、数据库配置

**MySQL 8.0** (默认)：`127.0.0.1:3306` / `mrp_system` / `root:root`

## 三、演示数据

13个物料 + 2个3层BOM + 18条MPS + 13条库存。MRP运算预期输出约77条计划订单。

## 四、核心测试用��（含新增功能）

### Excel导入物料+BOM一体化导入（重要！）
```bash
# 导入时自动创建物料表中不存在的物料
# 文件需包含两个Sheet：「BOM」+「物料主数据」
curl -X POST http://127.0.0.1:8000/api/bom/import/excel \
  -F "file=@BOM测试-蓝牙音箱.xlsx"
```
**预期结果：** `成功导入 1 个BOM，新建物料 15 个`

### BOM版本管理
```bash
curl -X POST http://127.0.0.1:8000/api/bom/version-bump/1 \
  -H "Content-Type: application/json" \
  -d '{"change_reason":"新版本测试"}'
curl -X POST http://127.0.0.1:8000/api/bom/activate/2
```

### BOM树形展开
```bash
curl http://127.0.0.1:8000/api/bom/tree/1   # FG-001 → 8 nodes
```

### MRP引擎 ⭐
```bash
curl -X POST http://127.0.0.1:8000/api/mrp/run \
  -H "Content-Type: application/json" \
  -d '{"horizon_days":60}'
# 预期: 77 orders, ~50ms, 20 exceptions
```

### MRP转订单
```bash
# 先MRP运算，再取结果转订单
curl -X POST http://127.0.0.1:8000/api/mrp/convert-to-orders \
  -H "Content-Type: application/json" \
  -d '{"planned_orders":[{"item_code":"RM-002","order_type":"PURCHASE","quantity":500,"required_date":"2026-07-10","release_date":"2026-07-03","level":2}]}'
```

### CRP产能计算
```bash
curl -X POST http://127.0.0.1:8000/api/crp/calculate \
  -H "Content-Type: application/json" \
  -d '{"horizon_days":56}'
```

### 到货检验
```bash
# 创建检验（需先有已下单采购单）
curl -X POST http://127.0.0.1:8000/api/inspection \
  -H "Content-Type: application/json" \
  -d '{"purchase_order_id":1,"item_id":7,"inspect_qty":500}'
# 录入结果（合格自动入库）
curl -X PUT http://127.0.0.1:8000/api/inspection/1/result \
  -H "Content-Type: application/json" \
  -d '{"pass_qty":490,"reject_qty":10}'
```

### 盘点
```bash
curl -X PUT http://127.0.0.1:8000/api/inspection/stock-counts/1/result \
  -H "Content-Type: application/json" \
  -d '{"actual_qty":195}'
curl -X POST http://127.0.0.1:8000/api/inspection/stock-counts/1/adjust \
  -H "Content-Type: application/json" \
  -d '{"operator":"管理员"}'
```

## 五、前端批量操作验证

| 页面 | 操作方法 |
|------|----------|
| 物料管理 | 勾选多行 → 批量停用 |
| MPS计划 | 「批量添加」按钮 → 按周自动生成；或勾选多行 → 批量删除 |
| 采购管理 | 勾选多行 → 批量下单/批量完成/批量删除 |
| 生产管理 | 勾选多行 → 批量下达/批量开工/批量完工/批量删除 |
| BOM管理 | 「导入Excel BOM」→ 选文件 → 自动建物料+建BOM |
