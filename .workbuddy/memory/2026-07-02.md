# Phase 2 — 核心页面 UI 重设计

## MrpRun.vue（MRP 运算 — 重点重设计）
- **数据就绪面板**：MPS/BOM/库存计数 + 缺少数据引导跳转
- **参数配置**：slider + input-number 双向联动，渐变大按钮 CTA
- **进度动画**：5 步骤进度条（展开BOM→计算净需求→生成订单→检测例外→汇总）

## MaterialList.vue（物料管理）
- 三级折叠视图视觉层次增强（箭头旋转动画、模块/零件计数徽章）
- 批量操作栏重构（分隔线 + 清晰操作分组）
- 搜索关键词标签显示 + 可关闭
- 表格价格格式化（`.toFixed(2)`）

## InventoryList.vue（库存管理）
- Pill 风格标签切换（4 个 Tab 按钮组）
- **库存进度条**：根据实际库存/安全库存比例渲染红/黄/绿色进度条
- 呆滞料预警摘要卡片（严重/关注/提醒 三色统计）
- 流水表格增强（数字颜色标识 + monospace 字体）

## BOM / 采购 / 生产 及其余页面（11 个文件）
- 批量修复硬编码 `#252526` / `#fafbfc` 浅色主题背景 → 设计 Token
- 所有页面 `.page-container` 统一为 `padding: 0;`（底层由全局 layout 负责）
- 每个页面 scoped CSS 从 10-50 行缩减至 1-2 行
